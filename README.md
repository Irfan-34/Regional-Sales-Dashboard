# 📊 Regional Sales Dashboard

An interactive **Power BI Dashboard** designed to analyze sales performance across products, channels, customers, and geographic regions. The dashboard provides actionable insights through KPIs, dynamic filters, visual analytics, and an intuitive dark-themed user interface.

---

## 🚀 Features

- 📈 Executive Overview with key business KPIs
- 💰 Revenue, Profit, Cost, and Profit Margin Analysis
- 🛒 Product & Channel Performance Insights
- 🌍 Geographic Sales Analysis with Interactive Maps
- 🎯 Dynamic Filtering by Product, Month, and Region
- 📊 Top Performing Products & Revenue Trends
- 🤖 Natural Language Q&A using Power BI Q&A
- 🔘 Blank Buttons for seamless page navigation
- 📑 Bookmark-based interactive navigation
- 🎨 Custom Dark-Themed Dashboard Design
- 📌 Cross-filtering and drill-down capabilities

---

## 📑 Dashboard Pages

### 🏠 Home Page
A centralized landing page providing easy navigation to all dashboard sections through interactive buttons.

### 📊 Executive Overview
- Total Revenue
- Total Profit
- Total Orders
- Total Cost
- Profit Margin Percentage
- Revenue and Profit Trend Analysis
- Dynamic Product, Region, and Month Filters

### 📦 Product & Channel Performance
- Revenue by Channel
- Profit by Channel
- Top Revenue-Generating Products
- Monthly Revenue Analysis
- Product Performance Comparison

### 🌎 Geographic Insights
- Revenue by Region
- Profit Margin by Region
- Interactive Sales Distribution Map
- Regional Performance Analysis

### ❓ Q&A Dashboard
Leverages Power BI Q&A to allow users to ask business questions using natural language and receive instant visual insights.
